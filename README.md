### Problem Statement
Write a Python program that takes a sentence from the user and reverses
each word in the sentence while keeping the word order the same.

---

### Example Input

### Output
I evol nohtyp


---

### Solution Description
The program reads a sentence from the user and splits it into words.
Each word is reversed individually using string slicing.
The reversed words are stored in a list and then joined back to form
the final sentence.
